基于python
安装ansible方式：
1.通过系统方式，yum、apt-get等
ubunt安装ansible
    apt-get install software-properties-common
    apt-add-repository ppa:ansible/ansible
    apt-get update
    apt-get install ansible
2.通过python的方式
  （推荐）python ./setup.py install
  easy_install ansible
  pip install ansible


第一次安装ansible，在/etc/下没有ansible配置文件，需要新建文件夹
命令：
  cd /etc
  mkdir ansible
  cd ansible
  vi ansible.cfg

ansible配置文件解释：
[defaults]
ask_pass #控制ansible 剧本playbook是否会开启自动默认弹出的密码
ask_sudo_pass #用户使用的系统平台开启了sudo密码的话，应该开启这一参数
gather_subset #设置收集的内容：包含all、network、hardware、virtual、factor、ohai
remote_port&remote_tmp&remote_user #客户机的设置,分别对登陆的用户和端口，以及临时目录
sudo_exe & sudo_flags & sudo_user #sudo命令相关设置,分别是sudo命令路径、sudo参数、能够使用sudo的user
forks  #最大开启的进程数，一般的设置方法是cpu核数*2
module_name #
valut_password_file #以脚本的形式将密码打印到标准输出中，脚本需要请求数据，请求将会发送到标准错误
#还有一些插件命令
[privilege_escalation] #执行命令的用户权限设置
[paramiko_connection] #paramika插件设置
[ssh_connection] #ssh连接设置
