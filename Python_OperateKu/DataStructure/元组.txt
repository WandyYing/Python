Python 的元组(tuple)与列表(list)类似，不同之处在于元组的元素不能修改。


a=(1,2,3)
a[1] #2 从元组中取值
a[1:]#(2,3)
元组也可以进行拼接

说下不同的地方：
元组使用小括号，列表使用方括号。
而且元组是不可变类型，这意味着不能在元组内删除或编辑任何值，可以通过del a 来删除a这个元组，这样元组a就不存在了
要创建只含有一个元素的元组，在值后面跟一个逗号
可以对任何一个元组执行拆封操作并赋值给多个变量，如x,y=(6,7) #x=6 y=7

元组内置函数：
len(a)
max(a)
min(a)
c=[1, 2, 3, 4, 2, 3, 5]
tuple(c) #c=(1, 2, 3, 4, 2, 3, 5) 将其他类型转换为元组
